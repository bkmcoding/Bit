### *A tiny bug in a vast, dreadful box.*

---

**[ PLAY THE DESCENT NOW ](https://bit.bkmcoding.com)**

---

### 🕷️ THE PREMISE
You are **bit**. You are small, fragile, and trapped. 
Every room is a box. Once you enter, the heavy doors slam shut. 
The swarm drops from the shadows. 
You must survive the wave, harvest mutations, and crawl deeper into the dark.

---

### ⌨️ CONTROLS
| Action | Input |
| :--- | :--- |
| **Move** | `W` `A` `S` `D` |
| **Aim** | `Mouse` |
| **Shoot / Spit** | `Left Click` |
| **Pause** | `ESC` |

*Opposite keys cancel (e.g., A+D). Use the environment to block projectiles. Never stop moving.*

---

### 🧬 MUTATIONS (UPGRADES)
Surviving a box grants you a choice to mutate your fragile form. 
Adapt to survive the deeper levels:
* **Adrenaline Gland:** Increased movement speed.
* **Thick Exoskeleton:** +1 Max HP.
* **Mandible Split:** Shoot projectiles in a V-shape.
* **Piercing Stinger:** Projectiles tear through multiple enemies.

---

### 🛠️ ARCHITECTURE & TECH
This game was heavily slopped under strict time constraints.
* **Engine:** (ReactJS) (Arcade Physics)
* **UI/Menus:** React & TailwindCSS
* **State Management:** Custom Global Entity Pool
* **Theme:** "Box" & "Spiders"

---

### ⚙️ LOCAL DEVELOPMENT
To build the box and run the game locally:

```bash
# Clone the repository
git clone [https://github.com/yourusername/bit.git](https://github.com/yourusername/bit.git)

# Navigate to the directory
cd bit

# Install dependencies
npm install

# Start the descent
npm run dev
```
